从零开始的奶妈级可调教铃心聊天机器人制作系列教程---安装篇

2021-11-04- 2021-11-16
1.5k- 4m-
bot教程

本文教你零基础从零开始搭建QQ聊天机器人的基础框架及插件

开篇废话

QQ聊天机器人能干什么？

  • 可以帮助你更方便的管理QQ群。
  • 可以添加各种娱乐功能丰富QQ群聊天内容。
  • 可以实现QQ群和各种游戏内容的对接。(如MC服务器管理,原神角色查询等等)
  • 对接各种实用接口进行信息实时查询。(天气信息,疫情通知等等)

没有任何基础的人想建立一个QQ聊天机器人困难吗？

  • 难,也不难,有许许多多的先驱者已经为我们铺设好了道路,多数功能我们只需要根据需要安装插件并修改配置即可

搭建一个QQ聊天机器人需要的开销大吗？

根据您对机器人的具体需求，这个开销可大可小，但几乎是免费或低消费的
对于搭建一个机器人总体花费需求只有两点，框架和运行平台，

  • 完全免费的：您可以把搭建好的机器人放在自己的电脑上(电脑关机=机器人关机),或者寻找愿意免费帮忙代挂的大佬
    或者想办法挂到学校服务器上白嫖资源
  • 低消费的：寻找愿意合租的人或前往淘宝租用共享服务器
  • 更加私有化的：前往某某云租用轻量型云服务器(1h2g就完全够用)

QQ聊天机器人的框架

目前常见的框架有酷Q 先驱 mirai 小栗子等等。本文用先驱框架举例子。你喜欢其他的框架？可以查看他们官方文档

  • 酷Q：愿我们能在一个更开放的平行世界相遇
  • 先驱(免费且推荐新手上手)：www.xianqubot.com
  • mirai(完全开源)：github.com/mamoe/mirai
  • 小栗子(付费但拥有不少精品插件)：xiaolz.cn

准备工作

本文使用windows系统及先驱框架作为基础，插件安装则以铃心自定义为例子，后续也会此插件为中心进行详细的教程

铃心自定义很大可能性上不会再更新了——至于它的替代品，我觉得也只有一些正经开发的道路能配得上了。
—–YoruNina

下载所需要的软件

  1. 我们前往先驱官网下载发行版框架本体
  2. 然后前往下载铃心自定义插件
  3. 一个闲置的QQ号(等级不要太低，最好也不要是那种长达半年未登录过的账号)

软件解压/插件安装

  1. 在桌面或者任意一个你觉得顺手的地方，创建一个新的文件夹
  2. 先将下载到的先驱框架解压进去 双击启动先驱.exe等待生成文件
  3. 将铃心自定义.XQ.dll解压至 plugin 文件夹内
  4. 此时的文件夹目录应该如下
    文件夹
    ├─先驱.exe
    ├─bin
    ├─Config
    ├─data
    ├─Plugin
    │ └─ 铃心自定义.XQ.dll
    └─Log
  5. 此时,机器人的基础文件配置已完成
  6. 双击先驱.exe选择账号管理,在页面内右键即可选择登录账号

插件开启及界面介绍

双击启动先驱.exe 进入框架主界面

  • 左侧由上至下依次是
  1. 框架日志 任何报错，公告，插件信息，机器人收发信息都会在此显示，找大佬提问时最好带上这一页的截图
  2. 账号管理 在这里登录机器人的账号，可以同时登录多个账号，但是不推荐
  3. 插件扩展 所有在plugin文件夹内放的.XQ.dll插件都会在这里显示，各个插件的菜单也通常从这里开启
  4. 框架设置 对于框架的各种设置，一般情况下无需修改
  5. 关于先驱/购买授权 显示框架的详细开发者等信息的地方，暂时属于无用按钮
  • 进行下一步之前，请先在账号管理内登录聊天机器人的账号

启用铃心并打开铃心设置页面

  • 单击框架扩展按钮,右键铃心自定义,选择启动插件，再次右键并选择设置插件，便可以打开铃心自定义的插件页面
  • 可以看到拥有十分多的功能，这些都还只是冰山一角
  • 插件设置分为了五个部分
  1. 主页 各种插件的基础设定，基础信息
  2. 基本设置 本插件的核心点，关键词的添/删/改的页面
  3. 进阶设置 设置定时任务/关键词分群/词库/积分对接等信息的页面
  4. 群管系统 设置黑名单/白名单等信息的页面
  5. 回复调试 快捷编辑/调试关键词的页面(推荐使用EPK编辑器来代替功能)

添加第一个关键词，让属于你的机器人向这个世界问好！

  • 进入基本设置页面，在空白处右击，选择添加
  • 在关键词内填写测试在内容里填写hello world
  • 直接关闭编辑框
  • 点击插件设置界面右上角的保存按钮
  • 点击插件设置界面右上角的保存按钮
  • 点击插件设置界面右上角的保存按钮
  • (点击一次即可，但请记住这个简单却又容易被忘记的步骤)
  • 前往QQ，把你和机器人拉一个小群，在群内直接发送测试

    恭喜，专属于你的机器人已经成功搭建
