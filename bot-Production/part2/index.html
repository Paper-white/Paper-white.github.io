铃心界面详解

主页

  1. 设置主人的QQ和群管功能触发词（主人权限为最高级权限）（mirai的铃心暂时无法使用主人权限）
  2. 插件接收信息设置 推荐打开详细日志和艾特转义
  3. 插件发送信息设置 推荐回复延迟设置为50~100毫秒 秒收发信息大概率会被tx风控
  4. 变量说明：说明书，多看！ 打开数据目录：【应用目录】的位置(后面会讲) ``

基本设置

  1. 四种显示列表
    • 关键词列表：默认列表，最常用的列表
    • 功能列表：导入包管理列表
    • 应用列表：导入包批量管理的列表
    • 分类列表：对关键词进行分类管理页面
  2. 关键词添加页面
  3. 匹配方式 触发方式
    • 完全匹配 所设关键词和发言一致时(区分大小写)
    • 模糊匹配 所设关键词包含于发言中(不区分大小写)
    • 前缀匹配 所设关键词是发言的前缀(区分大小写)
    • 正则匹配 所设关键词(即表达式)与发言相匹配时（匹配规则为完全匹配）
    • 详细匹配规则请参考变量说明
  4. 快捷添加特殊回复（有更好的函数代替，所以实际并不会经常使用的功能键）

面向群文件编程—EPK导入包的使用

什么是导入包？使用导入包都有什么优点缺点？

导入包即大佬们提前写好然后导出的关键词集
各种常用功能基本都有大佬有写好并分享出来的导入包
直接添加即可使用，可以短时间内就创建大量功能，十分方便
但是每个大佬的代码习惯都不一样，导致不同的导入包之间联动性很低，两个功能很难联系到一起

导入包如何使用，需要注意什么

在基本设置页面的右上角就可以找到导入按钮
导入包使用时注意导入包的备注，部分导入包作者会在备注里写上必要信息和简单的使用方法
下载导入包时注意导入包的发布时间，部分导入包使用的 网络接口(API) 可能失效导致导入包失效
导入后记得保存

如何删除导入包

在基本设置里右上角选择功能列表，选中功能右键点击删除即可
